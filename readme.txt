◆「謝辞」または「はじめに」
この度はMMF(Miku Miku Flex library)をダウンロードして頂き有難う御座います。
　　MMD(Miku Miku Dance)の登場、および様々な周辺ソフトの開発により動画制作環境はめまぐるしい進化を遂げ社会に貢献しています。　
　　MMD・PMDE・MME・プラグイン等各種ツールを公開してくださっている作者様に心からお礼申し上げます。
　　MMFではこれら先人達が築き上げてきた世界をさらに良いものとなるよう心から願いMMFライブラリを開発しています。
　　MMFのプロジェクトは開始されたばかりです。至らない点、皆さんにご迷惑をおかけする事もあるかもしれませが、今後ともご指導、御鞭撻のほどよろしくお願いいたします。

◆動作対象

　[最低動作環境]
　　Windows Vista SP2以降
　　DirectX10.1以上に対応したグラフィックカード
　[推奨動作環境]
　　Windows 7 64bit
　　DirectX11以上に対応したグラフィックカード

　　DirectX10.1までしか対応しない場合、テセレーションなどの一部機能が動作しません。

◆必須ランタイム

　.Net FrameWork 4.5(http://www.microsoft.com/ja-jp/download/details.aspx?id=3065
　DirectX End User Runtime(http://www.microsoft.com/ja-jp/download/details.aspx?id=35)
　SlimDX End User Runtime(http://slimdx.org/download.php)※

※　現在は、End User Runtimeの中の.Net4.0、x86を利用してください。MMFはx86版で開発されており、正式リリースのみx64版を同梱します。

◆ライセンス　　　
　このライブラリのライセンスはLGPLです。

このライセンスを満たす限り、断りなく利用してもらってかまいません。(商用利用などはMMF以外の権利者のライセンスに従ってください(モデル、モーションなど))
ただし、ライブラリとしても使用者を把握できたらうれしいので、何かいいものができたら是非報告を。

　MMDを取り巻くさまざまな技術的項目には、かつてから車輪の再開発が目立っていました。MMFは車輪の再開発を止めるために作成され始めたものです。
新たな仕様が策定され、MMFを改変し実装されてクローズドソースになればまた車輪の再開発が必要になります。

　よりよい仕様が思いついて実装したい際は、是非MMF開発グループへの参加を検討してください。それが嫌な場合、少なくとも改変部分についてオープンソースにしてください。
相談していただければそれが良い仕様であれば、開発チームも力を貸せます。
(PMXの仕様拡張などは行えません。これがプロジェクトで行われるとすれば極北PがPMXの仕様を変更したときです。)
