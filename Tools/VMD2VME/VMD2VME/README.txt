MMPとMMFで合同で策定している、拡張可能かつさまざまな言語から読み込めるモーション形式であるVMEファイルへのVMDの変換ツールです。
VME形式についてはhttps://github.com/sn0w75/VMEをご覧ください。
Python,C++,C#,VB,Java用のパーサーをVMEは提供していますが、ProtocolBufferのコンパイラがある言語であれば任意の言語で利用できます。

このツールの使い方
コマンドライン引数に対して、[第一引数]元のvmdファイル、[第二引数]変換先のvmeファイル　を与えてください。

なお、MMF,MMP共に近日中にVMEに対応する予定です。
